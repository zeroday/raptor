# Raptor-Web - Web Application Security Scanning (Alias)

Alias for `/web` - Scan web applications for OWASP Top 10 vulnerabilities.

## Usage

`/raptor-web [--url <url>] [--auth-token <token>] [options]`

## Examples

- `/raptor-web --url https://example.com`
- `/raptor-web --url https://myapp.com --auth-token "Bearer xyz"`

## Execution

This command is identical to `/web`. See `/web` command documentation for details.

Execute: `python3 raptor.py web --url <url> [options]`
