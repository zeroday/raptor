# RAPTOR Virtual Environment Usage

## Activation

To use RAPTOR with the installed virtual environment:

```bash
cd /path/to/raptor/raptor
source venv/bin/activate
```

Or use the full path:
```bash
/path/to/raptor/venv/bin/python raptor.py [command]
```

## Installed Tools

✅ **Python packages** (in venv):
- requests >= 2.31.0
- anthropic >= 0.39.0 (for Claude API)
- tabulate >= 0.9.0

✅ **Semgrep** (in venv):
- Version: 1.144.1
- Command: `./venv/bin/semgrep` or `semgrep` (when venv activated)

## Next Steps

1. **Set API Key** (required for LLM features):
   ```bash
   export ANTHROPIC_API_KEY=your_key_here
   # OR
   export OPENAI_API_KEY=your_key_here
   ```

2. **Optional Tools** (install as needed):
   - CodeQL: Download from https://github.com/github/codeql-cli-binaries
   - AFL++: `sudo apt install afl++` (for fuzzing)
   - GDB: Usually pre-installed, or `sudo apt install gdb`

## Usage Example

```bash
cd /path/to/raptor/raptor
source venv/bin/activate
export ANTHROPIC_API_KEY=your_key_here
python raptor.py agentic --repo /path/to/code
```
